# 📊 IPL Analysis Dashboard

This project is an interactive IPL Analysis Dashboard built using **Power BI** and **DAX** to analyze team and player performance.

## 🔹 Features
- 🏆 Tournament Winner Analysis  
- 📈 Top 5 Teams Performance  
- 🏏 Top Batsman & Bowler Statistics  
- 🟠 Orange Cap (Top Run Scorer)  
- 🟣 Purple Cap (Top Wicket Taker)  
- 📊 Bowler Economy & Strike Rate  

## 🔹 Tools Used
- Power BI  
- DAX  
- Excel  

## 🔹 Insights
- Identified **top performing teams and players**  
- Analyzed **winning trends across matches**  
- Compared **bowler and batsman performance metrics**
